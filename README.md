## Quick start (run locally)

1. Clone & enter the repo:

```bash
git clone <this-repo-url>
cd oceanicoder-portfolio-v2-fixed
```

1. Install dependencies:

```bash
npm install
# or
yarn
```

1. Start dev server:

```bash
npm start
# or
yarn start
```

1. Run tests / build:

```bash
npm test
npm run build
```
